# 🧬 Biogenios

> Aprende Biología resolviendo preguntas, no leyendo teoría.

Biogenios es una aplicación web progresiva (PWA) desarrollada para ayudar a estudiantes preuniversitarios a prepararse para exámenes de admisión mediante cuestionarios interactivos de Biología.

Actualmente se encuentra en etapa **MVP (Minimum Viable Product)** y está enfocada en validar la experiencia de aprendizaje antes de incorporar servicios en la nube.

---

## 🎯 Objetivo

Crear una plataforma rápida, accesible y gratuita donde cualquier estudiante pueda practicar Biología desde su celular o computadora, sin necesidad de registrarse.

El objetivo principal es que el estudiante dedique más tiempo a responder preguntas que a navegar por la aplicación.

---

## ✨ Características actuales

- 📱 Compatible con dispositivos móviles y computadoras.
- 🌐 Funciona directamente desde el navegador.
- ⚡ No requiere instalación (aunque puede instalarse como PWA).
- 💾 Guarda el progreso localmente en el dispositivo.
- 🏆 Ranking local.
- 📊 Estadísticas personales.
- 🎯 Cuestionarios interactivos.
- 🚀 Carga rápida.
- 🔓 Código abierto.

## 🚀 Ejecutar localmente

Clona el repositorio:

```bash
git clone https://github.com/USUARIO/biogenios.git
```

Ingresa a la carpeta:

```bash
cd biogenios
```

Abre `index.html` en tu navegador.

También puedes utilizar un servidor local como:

```bash
Live Server
```

o

```bash
python -m http.server
```

## 🌱 ¿Por qué existe Biogenios?

Muchos estudiantes no cuentan con plataformas accesibles para practicar preguntas de admisión desde cualquier lugar.

Biogenios nace con la idea de ofrecer una experiencia sencilla, rápida y centrada en el aprendizaje, aprovechando las tecnologías web para que estudiar sea tan fácil como abrir una página.

El proyecto evolucionará a partir de la experiencia y las necesidades reales de su comunidad de usuarios.
